# 闪蜂速运小程序 上线检查清单

- 项目类型：微信小程序前端交付包
- 部署方式：微信开发者工具 / 公众平台发布

## 1. 放行前

- [ ] QA 结论为 `PASS` 或 `PASS_WITH_RISK`
- [ ] 老板 / Controller 已确认本次上线窗口
- [ ] `08_go_live/DEPLOYMENT_PROFILE.md` 已补齐
- [ ] 域名、证书、端口、环境变量已确认
- [ ] 数据库迁移与回滚方案已确认

## 2. 执行前

- [ ] 记录 commit SHA / tag
- [ ] 备份数据库或上一版构建物
- [ ] 已通知相关负责人进入上线窗口
- [ ] 已执行或准备执行 `npm run validate:delivery`

## 3. 执行后

- [ ] 健康检查通过
- [ ] 核心主链路通过
- [ ] 监控和日志无明显异常
- [ ] 已记录实际发布时间和操作人
- [ ] 已记录剩余风险和观察窗口

## 4. 回滚条件

- [ ] 核心交易链路不可用立即回滚
- [ ] 数据写入异常立即回滚
- [ ] 域名、证书、鉴权错误影响用户访问立即回滚
