当前项目已经满足：

- 可导入微信开发者工具
- `npm run icons` / `npm run validate` 可作为前端交付校验
- 客户接入、接口合同、抖音部署、演示脚本等文档已存在

- 正式 API 域名未冻结

- `可演示`
