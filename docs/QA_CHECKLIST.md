# QA 验收清单

## 自动校验

```bash
npm run icons
npm run validate
```

## P0 主路径

| 编号 | 用例 | 预期 |
| --- | --- | --- |
| QA-001 | 首页点击“寄快递” | 进入寄件 tab |
| QA-002 | 首页点击“查快递” | 进入查件 tab |
| QA-003 | 运费时效试算 | 输入地址后费用和时效刷新 |
| QA-004 | 创建寄件单 | 生成运单并进入详情 |
| QA-005 | 查件 | 输入运单号后展示轨迹 |
| QA-006 | 我的寄件 | 新运单出现在列表顶部 |
| QA-007 | 我的页面 | 能进入地址簿、客服中心、运费时效 |

## P1 异常路径

| 编号 | 用例 | 预期 |
| --- | --- | --- |
| QA-101 | 空地址提交 | 提示填写寄件和收件地址 |
| QA-102 | 无运单查件 | 给出查询提示，不崩溃 |
| QA-103 | 订单详情返回 | 可回到订单列表 |
| QA-104 | 游客模式警告 | 识别为微信开发者工具限制，不当作业务错误 |
