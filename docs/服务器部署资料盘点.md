## 3. 这些资料怎么找

### 3.1 服务器侧

直接在服务器查：

```bash
uname -a
lsb_release -a
df -h
free -h
which nginx
which mysql
which docker
which node
which npm
which git
systemctl is-active nginx
systemctl is-active mysql
systemctl is-active docker
ss -lntp
```

这些命令可以找到：
- 系统版本
- 磁盘和内存
- 已安装运行环境
- 已占用端口

### 3.2 项目代码侧

在本地项目目录查：

```bash
cd /Users/xiaoliao/Desktop/openclaw/workspace/projects/shanfeng-express-miniapp
cat package.json
find . -maxdepth 2 -name '.env*'
rg -n "baseURL|api|secret|config|env" .
```

这些动作可以找到：
- 技术栈
- 构建命令
- 环境变量入口
- 配置文件位置

### 3.3 抖音开放平台

去抖音开放平台查：
- AppID
- AppSecret
- 管理员/协作者权限
- 服务器域名白名单
- 提审发布入口

当前已确认：
- AppID 已有
- 管理员权限已有

还要补：
- AppSecret
- 白名单配置

### 3.4 域名和证书

去运维或云控制台查：
- 域名归属
- DNS 解析权限
- 证书是否已有
- 证书是否可复用

常见落点：
- 腾讯云 / 阿里云 DNS
- Nginx 配置目录
- `.acme.sh`

### 3.5 数据库

去 DBA / 运维或服务器查：
- 是否允许新建库
- 是否允许新建账号
- 现有 MySQL 版本

如果有权限，可以进一步查：

```bash
mysql --version
```
