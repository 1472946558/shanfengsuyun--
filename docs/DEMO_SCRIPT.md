# 演示脚本

## 目标

向客户演示 `闪蜂速运` 小程序的完整寄件闭环：进入首页、创建运单、查询轨迹、查看订单、进入个人中心。

## 演示流程

1. 打开首页，说明品牌定位：企业级同城与快递寄件服务。
2. 点击 `寄快递`，进入寄件页。
3. 填写寄件地址、收件地址、服务产品、托寄物、重量、保价和备注。
4. 点击 `确认寄件`，生成模拟运单。
5. 进入 `我的寄件`，查看刚创建的运单。
6. 打开运单详情，展示费用、距离、预计时效、收派员和轨迹入口。
7. 进入 `查件`，输入运单号，展示轨迹时间线。
8. 回到首页，展示 `运费时效`、`服务网点`、`地址簿`、`客服中心`。
9. 进入 `我的`，展示账户资料、常用功能和企业寄件入口。

## 讲解重点

- 当前版本是可演示前端，已封装 RESTful API 合同。
- 客户接入服务器后，可以替换本地 mock 为真实订单、轨迹、网点和客服数据。
- 品牌、图标、文案、页面结构均为原创，不依赖第三方快递品牌素材。

## 验收口径

- 页面可完整打开。
- 寄件流程能生成运单。
- 查件流程能展示轨迹。
- 底部导航和功能入口可点击。
- 页面不显示开发错误信息。
- 正式上线前必须接入真实后端、域名、隐私协议和资质。
