# PRD：闪蜂速运小程序

## 项目目标

打造一个原创品牌的快递/同城配送小程序，覆盖个人寄件、企业寄件、查快递、运费时效、地址簿、服务网点、订单中心和客服售后。

## 核心业务链路

1. 用户进入首页。
2. 选择寄快递或运费时效。
3. 填写寄件地址、收件地址、服务产品、托寄物、重量、预约上门和保价。
4. 系统试算费用和时效。
5. 用户提交寄件单。
6. 系统生成运单。
7. 用户在查件页输入运单号或查看最近运单。
8. 用户在订单页查看历史寄件。
9. 异常时进入客服中心创建工单。

## 商用边界

当前版本为前端可演示交付包。正式商用需要客户补齐后端服务、合法域名、微信登录、手机号授权、隐私协议、地图服务、支付、消息通知、客服工单、运营资质和安全风控。
